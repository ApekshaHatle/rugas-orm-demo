### 1) Clone the repository:
 ```
git clone https://github.com/ApekshaHatle/rugas-orm-demo.git
```
```
cd rugas-orm-demo
```
### 2) Setup .env in ALFREDTASK folder
```
MONGO_URI=...
PORT=...
JWT_SECRET=...
NODE_ENV=...
CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...
```
### 3) Install dependencies
```
npm install
```
```
cd frontend
npm install
```
### 4) Run the Project
```
cd frontend
npm run dev
```
```
cd ..
npm run dev
```
